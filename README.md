# Unreal-C
Unreal C version 1.0.0

Features in this version:\
1- int, float and complex numbers\
2- bool and none types\
3- strings\
4- lists, tuples, dicts and sets\
5- binary and unary operators (+, -, *, /, %, ==, != and ...)\
6- veriables\
7- if statement\
8- switch statement
